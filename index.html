Czym jest front-end?

Front-end w skrócie jest to wszystko, co widzisz na stronie internetowej.

Składają się na niego trzy języki: HTML, CSS oraz JavaScript. Na tej stronie zobaczysz wiele informacji na temat tych technologii.

Tekst, który widzisz jest widoczny dzięki HTML. Jest on ułożony w taki sposób dzięki CSS. A JavaScript? Zobaczysz jak przejdziesz dalej :)

HTML

Jest to hipertekstowy język znaczników. Wiele osób myli HTML z językiem programowania, którym on w żadnym wypadku nie jest.

Określa się w nim zawartość strony taką jak tekst czy zdjęcia.

Zobacz, co można zrobić w HTML klikając przyciski poniżej.

CSS

CSS jest to język reguł stylizacji. To dzięki niemu strony teraz wyglądają inaczej niż w latach 90.

W CSS określa się style stron, układ tekstu, sekcji czy zdjęć. Określa on również kolory czy czcionki tekstów.

Kliknij przyciski poniżej aby zobaczyć co można zmienić w CSS.

JAVASCRIPT

JavaScript jest to skryptowy język programowania. Dzięki niemu strony są dynamiczne oraz interaktywne

Płynna animacja przewijania i przyciski w poprzednich sekcjach są zasługą tego języka. Bez niego strony
internetowe byłyby nudne a użytkownik mógłby tylko przeglądać treść, która też byłaby okrojona.
Otwarzacz na Youtube, Mapy Google i wszystkie animacje są stworzone w JavaScripcie. Jest on najpotężniejszą
częścią front-endu.
